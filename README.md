# resume-template
latex简历模板，可在Overleaf中编译-下载pdf版本

步骤：
- 把resume目录下载到本地，并压缩成zip格式
- 打开Overleaf:https://www.overleaf.com/project
- 点击左上角「New project」->「Upload project」->「选择压缩好的resume.zip文件」
- 导入后点击左上角「File」->「Settings」
- 弹出窗口左侧菜单栏选择「Complier」后在右侧Complier中选择「XeLaTex」编辑器
- 根据自己的信息修改「main.tex」中的内容
- 点击「Recomplie」编译即可得到pdf版本，点击下载即可
